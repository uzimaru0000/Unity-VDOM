# Veauty

Implementation of Virtual-Dom for Unity3D

## Environment

- Unity3D 2019.1 or higher
- C# 7.0

## Install

Add line in `Packages/manifest.json`

```json
{
  "dependencies": {
    ...
    "com.uzimaru.veauty": "https://github.com/uzimaru0000/Veauty.git",
    ...
  }
}
```
